## Create a project

```bash
git clone https://github.com/puncsky/mobile-onefx-boilerplate.git my-awesome-mobile-project
```

## Run your project

```bash
cd my-awesome-mobile-project

nvm use 10.15.0
npm install
```

### Development mode
To run your project in development mode, run:

```bash
npm run start
```
